## Обзор

Полный редизайн визуальной части приложения Яндекс.Дом от Apple-подобного светлого стиля к современному интерфейсу с градиентным фоном и полупрозрачными элементами с блюром.

## Изменения

### 1. Удалён Toolbar
- Компонент `Toolbar.tsx` полностью удалён
- Кнопки настроек (тема, автозапуск, обновить, информация, выход) перенесены в правый верхний угол контентной области
- Кнопки отображаются **только** на странице "Все устройства" (`activeSidebarView === 'home'`)
- Кнопка редактирования дашборда также доступна только на странице "Все устройства"

### 2. Градиентный фон
- Весь фон приложения — градиент `linear-gradient(135deg, #176f91, #20aac4)`
- Применён на `body`, просвечивает через все прозрачные элементы
- Тёмная/светлая тема больше не влияет на фон, но продолжает работать для модальных окон

### 3. Прозрачный сайдбар
- Фон сайдбара: `rgba(255, 255, 255, 0.06)` с `backdrop-filter: blur(20px)`
- Текст: белый с разной прозрачностью (`rgba(255, 255, 255, 0.9)` для основного, `0.45` для заголовков секций)
- Выпадающий список домов: тёмный полупрозрачный с блюром

### 4. Индикация состояния устройств (вместо свитчера)
- iOS toggle (`.ios-toggle`) удалён из DeviceCard
- **Выключенное устройство**: прозрачная карточка с `backdrop-filter: blur(12px)`, белый текст, белая иконка
- **Включенное устройство**: белая карточка, чёрный текст, зелёная иконка (цвет `var(--toggle-on)`)
- **Карточка группы**: всегда прозрачная с `backdrop-filter: blur(8px)`, все кнопки (включение, настройки, избранное) сохранены
