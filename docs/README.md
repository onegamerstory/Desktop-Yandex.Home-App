## О проекте

**Yandex Smart Home Control** — десктопное приложение на Electron + React 19 для управления устройствами умного дома Яндекса. Работает через Yandex IoT API, обеспечивает включение/выключение устройств, управление сценариями, группами, просмотр камер и мониторинг датчиков.

Данная папка `docs/` содержит техническую документацию, предназначенную для разработчиков и архитекторов, поддерживающих и развивающих проект.

## Формат документов

Все документы написаны в едином стиле:

- **Frontmatter:** блок метаданных (версия документа, дата, назначение, аудитория)
- **Оглавление:** навигация по якорным ссылкам в начале каждого документа
- **Заголовки:** `##` — основные разделы, `###` — подразделы, `####` — шаги/подподразделы
- **Таблицы:** выровненные, с заголовками в жирном начертании
- **Код:** блоки с указанием языка (typescript, js, text) для подсветки синтаксиса
- **Связанные документы:** ссылки на смежные файлы в конце каждого документа
